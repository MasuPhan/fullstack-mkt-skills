## 3. ROAS & ROI

```
ROAS = Doanh thu / Chi phi quang cao
  > 3x: Tot
  > 5x: Xuat sac
  < 2x: Can toi uu gap

ROI = (Doanh thu - Tong chi phi) / Tong chi phi Ă— 100%
  Tong chi phi = Ads + Content + Nhan su + Tool + UGC
```

### CPMess (Chi phi / Tin nhan)
